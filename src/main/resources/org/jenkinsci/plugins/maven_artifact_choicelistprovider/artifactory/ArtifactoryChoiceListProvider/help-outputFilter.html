Optional: The artifacts list can be processed by setting this parameter.

Parameter takes numbers separated by |

The repository URL will be split by "/" and you can specify the index of each chunk which should be present in the select box.

Defaults:

  • delimiter: ,
  • split: /

For example: {"sort":"-2"}

For example: {"split": "/", "sort":"-2", "delimiter":"|"}

For example: {"split": "/", "sort":"-2", "delimiter":"|", "output":[-2, -3]}

For example: {"split": "/", "sort":"-2", "delimiter":"|", "output":["-2", "-3"]}
